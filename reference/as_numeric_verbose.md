# Convert to Numeric with Warnings Converts input to numeric, reporting non-convertible values (for debugging) or transparency.

Convert to Numeric with Warnings Converts input to numeric, reporting
non-convertible values (for debugging) or transparency.

## Usage

``` r
as_numeric_verbose(x, quiet = FALSE)
```

## Arguments

- x:

  Input vector.

- quiet:

  Logical; if FALSE, prints non-convertible values.

## Value

A numeric vector.
